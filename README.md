# Origami

> Some cool scss stuff

## Development setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm start
```

## Build

```bash
# generate static project for production create folder /public
$ npm build
```

## File structure

```bash
.
├─ assets
├─ sass
│   ├─ components
│   ├─ layout
│   ├─ pages
│   ├─ reset
│   ├─ utilities
│   └─ main.scss
└─ index.html
```
